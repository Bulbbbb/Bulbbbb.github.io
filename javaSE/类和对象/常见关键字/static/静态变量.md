+ _static 只能修饰类的成员变量，不能修饰方法里的局部变量。_

### 静态变量

#### 定义

#### 用法

+ _为员工设置 id_

```java 
class Employyed{
	private static int nextId=1;
	private int id;
	
	public void setId(){
		id = nextId;
		nextId++;
	}
}
// 为harry设置id
harry.setId()
```

+ _统计类的实例数_

```java
public class MyClass {
    private static int instanceCount = 0;

    public MyClass() {
        instanceCount++;
    }

    public static int getInstanceCount() {
        return instanceCount;
    }
}

```
