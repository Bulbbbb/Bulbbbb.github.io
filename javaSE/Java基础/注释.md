## 注释分类

### 单行注释

_以 "//" 开始，表示从 "//" 到行尾的内容都是注释。_

```java
// 这是一个单行注释
int x = 10; // 这也是一个单行注释
```

### 多行注释

_以 "/\*" 开始，以 "\*/" 结束，表示 "/\*" 和 "\*/" 之间的所有内容都是注释。_

```java
/*
这是一个多行注释
它可以跨越多行
*/
```

### 文档注释

_以 "/\*\*" 开始，以 "\*/" 结束，表示 "/\*\*" 和 "\*/" 之间的所有内容都是文档注释。_
_文档注释是一种特殊类型的注释，它可以用于自动生成文档，包括类、方法、变量等的说明。_

```java 
/**
 * 这是一个文档注释
 * 它可以包含多行
 */
public class MyClass {
    /**
     * 这是一个文档注释
     * 它用于说明方法的作用
     */
    public void myMethod() {
        // ...
    }
}
```
